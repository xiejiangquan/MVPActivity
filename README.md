# MVPActivity

自动构建mvp，

1、下载到本地  git clone https://github.com/xiejiangquan/MVPActivity.git

2、将将MVPActivity整个文件夹拷贝到: <androidStudio-folder>/plugins/android/lib/templates/activities文件夹下

3、重启android studio

4、新建activity New->Activity->MVPActivity
